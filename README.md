# 🧠 Gimnasio Mental en Python

⏱️ Duración diaria total: 1 hora  
📍 Momento fijo: Justo después de bañarme y vestirme  
🎯 Objetivo: Recuperar lógica, fluidez, disciplina y confianza  

---

## 🔥 Día 1: Fundamentos + Ejercicios básicos

### ⏱️ 0:00 – 0:10 (10 min) → Calentamiento
Repasar if / else
Repasar for y while
Repasar listas y métodos básicos

### ⏱️ 0:10 – 0:40 (30 min) → Ejercicios
Sumar todos los elementos de una lista
Encontrar mayor y menor
Contar cuántas veces aparece un número
Sumar solo números pares
Invertir una lista

### ⏱️ 0:40 – 1:00 (20 min) → Reto del día
Encontrar el número más repetido en una lista

# 📌 Reglas del Gimnasio

Programar siempre justo después de vestirme
Intentar mínimo 10 minutos antes de buscar ayuda
Terminar siempre el reto final
Registrar lo aprendido en 3 líneas
Repetir ciclo aumentando dificultad
